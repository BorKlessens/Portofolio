Persoonlijk Project - Front-end Development

De opdracht: (Persoonlijk Project)

Tijdens dit project is het de bedoeling dat we gaan ontdekken welk vakgebied van Media Design je het meeste ligt. Je mag dus zelf bepalen wat je gaat doen en het is vooral de bedoeling dat je gaat oefenen om een goede stap richting het volgende semester te maken.

Proces

Netflix Project

Om te beginnen moest ik eerst bepalen wat ik wilde doen. Aangezien de kant van het Front End Developen mij het meest aansprak besloot ik om gewoon te gaan oefenen door steeds kleine projectjes te maken in VS Code. Als eerst besloot ik om een nieuwe pagina te maken voor Netflix. Dit omdat ik nog een beetje wil oefenen met het schrijven van Java Script en CSS animaties. Voor deze Netflix pagina ben ik begonnen met het maken van een geanimeerde slider met de populairste films/series. Daarna ben ik begonnen met de tweede slider. Hier heb ik verschillende afbeeldingen van film/series gelinkt en een mooi effect aan gegeven om de pagina nog wat meer op te vullen. Ook heb ik een header gemaakt voor de pagina en de hele stijl in de kleuren van Netflix gehouden. persoonlijk-project-3dslider-9hu10tnys-borklessens-projects.vercel.app Vercel: https://github.com/BorKlessens/Persoonlijk-Project.git

3D Web Design Onderzoek

Aangezien ik ook graag iets met 3D designs in mijn website wilde doen ben ik hier verder onderzoek naar gaan doen. Dit heb ik gedaan door verschillende YouTube filmpjes te kijken over hoe dit in z'n werking gaat. Ik ben begonnen met filmpjes over Blender (https://www.youtube.com/watch?v=Z8sg0nHNTTo). Het idee was namelijk om zelf 3d designs in stijl van mijn website te maken. Echter kwam ik erachter dat het leren van Blender een heel project op zich is aangezien ik hier de tijd niet voor had ben ik over gegaan in het invoegen van bestaande 3d designs. Ik had gelezen dat je dit het beste kon doen met Three JS, dus ben ik hier weer verder onderzoek naar gaan doen (https://www.youtube.com/watch?v=wxvSHOrBHbw, https://www.youtube.com/watch?v=Q7AOvWpIVHU&t=11s). Lang prutsen besloot ik het op een andere makkelijke manier te doen. Ik kwam op de website "Spline" Hier heb ik een aantal bestaande 3d designs van gehaald en door dit weer te linken door een simpele script regel bovenaan de code. @3D Splines

Website Design Inspiratie Project

Nadat ik hier en daar een beetje geoefend had was het tijd om te beginnen aan het grote project. Hiervoor kwam ik met een idee om een website te maken waar verschillende designs voor het maken van een website op staan. Ik loop namelijk zelf nog wel is aan tegen het feit aan dat ik geen inspiratie heb voor mijn website. Ik heb uiteindelijk 4 verschillende designs gemaakt. Daarna ben ik verder gegaan met het maken van de homepagina waar je kan kiezen uit de verschillende designs. (Zie 3d website in Github > https://github.com/BorKlessens/Persoonlijk-Project.git) persoonlijk-project-bxf5-f1wfbpwsi-borklessens-projects.vercel.app

Blender Projecten

Als laatste had ik als doel om het programma Blender te leren zodat ik dit later weer kan toepassen in toekomstige opdrachten. Hiervoor heb ik enkele tuturials voor gevolgd op YouTube waarin exact verteld werd wat ik moest doen en welke keybinds ik moest gebruiken etc. Ik heb een aantal van deze tutorials gevolgd om wat beter te worden in het werken met Blender. Uiteindelijk zijn hier 3 mooie projecten uitgekomen @Blender projecten

Reflectie

Het doel van dit project was om verder te gaan met het vakgebied wat je het meest aan sprak binnen Media Design en om hier verder mee te oefenen. Ik denk dat ik dit doel goed bereikt heb en een goede stap richting het volgende semester heb gemaakt als het gaat om de talen HTML, CSS en JavaScript. Dit is een beetje de basis die je nodig gaat hebben voor het volgende semester en deze beheers ik nu erg goed.

Ook heb ik veel geleerd over blender en dat gaat in de toekomst zeker van pas komen in toekomstige opdrachten. Al met al denk ik dat ik heel veel heb geleerd en goede stappen heb gezet in het verbreden van mijn kennis.
